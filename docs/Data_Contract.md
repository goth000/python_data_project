# Data Contract

## Week 1

На первой неделе источник данных ещё не выбран окончательно.

Планируемая структура проекта по данным:

- RAW — сырые данные из API;
- NORMALIZED — очищенные и нормализованные таблицы;
- MART — витрина данных с агрегатами и KPI;
- BI — отчёт или дашборд.

Подробная схема данных будет заполнена на следующих неделях.
